About

Bus Reservation System with Website is a very wonderful Bus and fleet Booking System, This Bus reservation system is developed by PHP and CodeIgniter - 4 .It is mostly useful for any Bus, Fleet and Ticket Selling Company.It has a variety of features on Bus ticket Management.

Key Features of BUS-Transport

  1. The Admin Panel dashboard is a hub where admins can effortlessly get information about:
    • Total active trip for currentday.
    • Total ticket booking for currentday.
    • Total booking amount for today.
    • Total passengers for todays trip.
    • Total inactive users in the marketplace.
    • Yearly income and expencse in bar & line chart
    • Weekly income and expense
    • Total transaction of payment methods
    • Monthly income and expense
    • Monthly total ticket booking
  2. Admin can book,cancel,refund any ticket which is booked user admin or agent. Admin also can see the journey list for any trip. Unpaid ticket can be automatically remove form system after a short period of time which can edited or update by the admin.
  3. Admin can add,edit,update,delete any agent in the system and also can see all transaction and commission of any agent.
  4. All type of accoount transacton can be maitained by admin. Admin can also edit,delete,update any transaction in the system
  5. All new passengers can be add to the system by the admin and the details of a passenger can be edited any time by the admin.
  6. Any type of employee role can be create by the admin and add different users against the employee role. later the role and the employee details can be edited.
  7. Admin can access all type of report generated in the system . He can check the total booking list, booking amount, trip lost angains a data or trip route.
  8. Any type of inquiry from users can be access by the admin. Admin also can add fleet type for any vehicle and its fitness period. Then he can assing different vehicles to different fleet type. Later all option can be edited by the admin.
  9. Admin can add different location and time for a trip. He can add trips as much he needs later a trip can be devided in many sub-trips. Separate journey fee also can be added to every trip and sub-trip.
  10. Different type of tax can be added by the admin, later all the tax amoount will be autmatic add to any transaction. Admin can add different type of Payment method to the system.
  11. Coupon for diffent trip and sub trip with date range can be add,delete,update by admin.
  12. A system need user to maintain so admin can add different type of role for the system later permission for that role can also be edit by the admin.
  13. All the front end part of the website can be editd my the admin because all the front end is maintain by api
  14. A blog section is install in the system so all type of blog can be add,edit,delete by the admin user.
  15. All the content of the system pages can be edited by the admin
  16. Admin can add different type of language and its strig values in the system. Automatically send mail & social media share can be configure by the andmin.
  17. Hole website configaration including button,text color can be maintain by the admin panel. Logo, faveicon, Logo-text, title font timezone country can be added by the admin.
  18. Admin can take backup of his system database any time.

Requirements for BUS Transport System

In this section we will present the recommended configuration under which BUS Transport would work perfectly. These are not absolute values as the performance depends on many factors such as host performance, server configuration and the number of visitors.

Server requirements

  • Apache
  • PHP version 8.0 (ie. recommended: PHP version 8.0)
  • MySQL version 5.0 or greater (recommended: MySQL 5.5 or greater)
  • Node.js version 16.15.0 (LTS)
  • We recommend DigitalOcean for hosting or any linux server

BUS Transport Installation

  1. Please extract the BUS Transport.zip folder. You will then get the following:

    • BUS Transport.zip
    • Documentation (Folder)

    We highly recommend that you read the documentation first.

  2. Extract BUS Transport.zip file to your Computer.
  3. Navigate into the extracted folder you will find 2 folder backend and frontend. Go to the front end folder.
  4. You will see .env file there.

  5. Open the .env file with any text editor and replace the content according to below images.

  6. If You don't have node install in your device, then do the following steps as the bellow images shows.
  7. Open any browser and type "node install" and do the following

  8. Navigate into the extracted folder. Go to the frontend folder, and run Terminal/Cmd, (i) Type "npm i --legacy-peer-deps" after that (ii) Type "npm run build"

  9. After finishing the process there will be a new folder name build. Go in to the build folder you will find some build files (eg. index, menifest, Static etc)

  10. Copy all the file and folder from build folder and paste it into root directory where backend and frontend folder are located.

  11. Upload all the folders and files execpt ("frontend folder") to server root directory ie. into the public_html directory.
    CAUTION: Make sure that the .htaccess file is moved correctly or else you will be unable to access the admin panel.

    12. NOTE: The instructions above will be applicable for all other system!


  12. After giving user name and password press singin button. if following error occer then give the database groupby permissin.

  13. Open Php MyAdmin from the root directory press SQL button from the top menu bar.

  14. A Sql Box will be open.

  15. Then Your Project is ready to run go to login screen with this url http://yourdomain.com/backend/login .
